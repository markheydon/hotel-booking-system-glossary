Cancel Booking

Description

A booking is cancelled.

  1. A Guest or Receptionist requests to cancel a booking.
  2. The user enters the booking reference number or the dates and guest name to find the booking.
  3. The system retrieves the booking.
  4. The user confirms they want to cancel the booking.
  5. The system cancels the booking by de-allocating the rooms (changes the status to 'CANCELLED' on the Booking).
  6. The booking is cancelled and the user is informed.
