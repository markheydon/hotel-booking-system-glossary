RoomType_Room

Description

RoomTypes (within a Hotel) will have one or more Rooms of that type which can be Booked.  A Hotel could have zero Rooms of a specific RoomType but then Bookings would be able to made for any rooms.  I.e. it is insufficient to have a RoomType without any Rooms in a working system although technically possible.
